# Apps

Deployable applications: the Next.js web app, the NestJS API, the FastAPI sandbox/execution service, and the content-engine worker (see [`docs/10-frontend/overview.md`](../docs/10-frontend/overview.md), [`docs/11-backend/overview.md`](../docs/11-backend/overview.md)).

## Anticipated apps

- `apps/web` — the Next.js/React frontend (lesson runner, knowledge graph explorer, AI Tutor surface)
- `apps/api` — the NestJS API (auth, content serving, progress, knowledge graph)
- `apps/sandbox` — the FastAPI sandboxed Python execution service
- `apps/worker` — the BullMQ-based content-engine/living-curriculum background worker

## Status

Empty by design. Per [`CLAUDE.md`](../CLAUDE.md), no application code is scaffolded here until Phase 0's documentation and schemas are substantially complete and the roadmap explicitly calls for it (see [`docs/20-roadmap/overview.md`](../docs/20-roadmap/overview.md) Phase 1).
