# Projects

Weekly/mini projects and "Boss Battle" milestone projects (see [`docs/00-foundations/PROJECT_GLOSSARY.md`](../docs/00-foundations/PROJECT_GLOSSARY.md)), distinct from routine in-lesson labs (`labs/`) in that they synthesize a full skill folder's concepts rather than a single one.

## Structure per project (once authored)

- Problem brief framed against the AI Energy OS mission
- Less-scaffolded than a lab — the point is synthesis, not step-by-step guidance
- Explicit link to which piece of the capstone architecture ([`docs/07-capstones/overview.md`](../docs/07-capstones/overview.md)) it contributes to
- Rubric-based assessment criteria (see [`docs/05-assessments/overview.md`](../docs/05-assessments/overview.md))
