# 05 — pandas

**Mission:** Manipulate Data (Mission 2)

## Capstone contribution

Consumption and generation data analysis — turning raw ingested data (from SQL + APIs) into the feature tables forecasting models will consume.

## Learning objectives

- DataFrames, indexing, filtering, aggregation
- Time-series-specific operations (resampling, rolling windows) — directly relevant to demand/generation data
- Merging/joining datasets from multiple sources (weather + consumption + price)
- Data cleaning for real-world, messy energy datasets
