# 06 — scikit-learn

**Mission:** Predict the Future (Mission 3)

## Capstone contribution

Demand and generation forecasting — the first real "Predict" layer of the AI Energy OS (see [`docs/00-foundations/MISSION.md`](../../docs/00-foundations/MISSION.md) Layer 1).

## Learning objectives

- Supervised learning fundamentals: regression and classification
- Time-series forecasting techniques and evaluation (avoiding leakage from the future — a common, mission-critical mistake in energy forecasting)
- Feature engineering from the pandas-prepared datasets in the prior skill folder
- Model evaluation, cross-validation appropriate for time-series data
- Baselines: why a naive forecast is a required comparison point, not a strawman
