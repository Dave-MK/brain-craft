# 07 — PyTorch

**Mission:** Teach Machines (Mission 4)

## Capstone contribution

Deep forecasting models where classical scikit-learn approaches plateau — sequence models for demand/generation forecasting.

## Learning objectives

- Neural network fundamentals: layers, activations, loss functions, backpropagation, optimizers
- Building and training networks in PyTorch, not just calling a pre-built model
- Sequence models relevant to time-series forecasting (RNN/LSTM/temporal conv/attention, as appropriate)
- Diagnosing training failures (vanishing gradients, overfitting, bad learning rates) as a practical skill, not just theory
- Honest comparison against the scikit-learn baseline from the prior skill folder — deep learning is adopted only where it earns its complexity
