# 04 — APIs

**Mission:** Manipulate Data (Mission 2)

## Capstone contribution

Weather and electricity price data ingestion — the AI Energy OS's forecasting layer needs live, external data, not just static seed files.

## Learning objectives

- HTTP fundamentals, REST conventions
- Authentication (API keys, OAuth basics)
- Consuming a real weather API and a real energy-price API
- Handling rate limits, pagination, and unreliable third-party data gracefully
