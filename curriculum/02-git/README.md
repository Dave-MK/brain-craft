# 02 — Git

**Mission:** Learn Programming (Mission 1)

## Why this skill, second

Because everything built from this point on — labs, the digital twin, the eventual AI Energy OS — is real, versioned software. Engineers collaborate and track history; this is a working habit, not an academic topic.

## Capstone contribution

Version control discipline applied to every subsequent artifact — this skill doesn't produce a standalone piece of the AI Energy OS, it's the practice that makes the rest of the build trustworthy and reviewable.

## Learning objectives

- Commits, branches, merges, rebasing basics
- Reading and writing meaningful commit history
- Resolving conflicts without panic or data loss
- Collaborating via pull requests (even solo, as practice for future contributors)

[`01-python`](../01-python/README.md) (something worth versioning)
