# 01 — Python

**Mission:** Learn Programming (Mission 1)

## Why this skill, first

Every later skill folder assumes working Python. Python is also the language the AI Energy OS's forecasting, data-processing, and RL work will ultimately run on.

## Capstone contribution

Weather/consumption data parsers, general scripting glue used by every later mission. The digital-twin seed data pipeline starts here.

## Learning objectives (high-level — full concept list lives in the knowledge graph, [`docs/08-knowledge-graph/overview.md`](../../docs/08-knowledge-graph/overview.md))

- Core syntax: variables, control flow, functions, data structures (lists, dicts, sets)
- Files and I/O — reading real CSV/JSON energy/weather datasets, not toy files
- Modules, packages, virtual environments
- Error handling and debugging fundamentals
- Enough OOP to read/write the codebase this platform (and the twin) will use

## Prerequisites

None — this is the entry point.
