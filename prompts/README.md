# Prompts

Reusable, versioned prompt templates for the AI systems this platform runs on:

- **AI Tutor** prompts (Socratic questioning behavior, see [`docs/03-ai-tutor/overview.md`](../docs/03-ai-tutor/overview.md))
- **The Architect** prompts (adaptive sequencing decisions, see [`docs/04-adaptive-learning/overview.md`](../docs/04-adaptive-learning/overview.md))
- **Content generation** prompts (living-curriculum lesson authoring/regeneration, see [`docs/09-content-engine/overview.md`](../docs/09-content-engine/overview.md))
- **Assessment grading** prompts (rubric-based evaluation of open-ended work, see [`docs/05-assessments/overview.md`](../docs/05-assessments/overview.md))

## Convention

Each prompt template is versioned and dated, since prompt changes affect tutoring/grading behavior the same way a code change would — treat them with the same review discipline as application code, not as throwaway strings.
